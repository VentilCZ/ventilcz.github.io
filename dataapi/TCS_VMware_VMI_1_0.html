Parameter Mandatory Purpose Target attribute Attribute type Description
attr01 true Name (name) String (255) The name of the CI. It should uniquely identify the CI for a human. This is not forced as a unique field on database level, but set as unique on the user interface for defined CI classes (Server, Network Gear etc).
attr02 true Correlation ID (correlation_id) String (512) External ID of the CI used in the external primary data source provider
attr03 false Installed (install_date) Date/Time Timestamp when CI first time taken in use
attr04 false Description (short_description) String (1,000) Optional free text to describe important aspects of the CI, information that is not possible to store in attributes / KB and other related CIs via relationships. This attribute is visible to customers.
attr07 false FQDN (fqdn) String (255) Fully Qualified Domain Name associated to the CI
attr08 false IP Address (ip_address) String (255) Primary production IP address
attr09 false Primary Business Service (u_primary_bs) Reference (Automated Business Service) CI number or Sys ID corresponding Business Service (SO)
attr10 true Lifecycle (install_status) Integer Lifecycle shows current CI lifecycle phase. See the linked knowledge article for more information.
attr11 false Provider SO (u_provider_so) Reference (Service Offering) CI number or Sys ID corresponding Factory Service Offering
attr13 false Cost center (cost_center) Reference (Cost Center) Reference to Tieto internal Cost center
attr14 false CPUs (cpus) Integer Number of virtual CPUs multiply by virtual cores
attr16 false Disks size (GB) (disks_size) Integer Total provisioned storage for a VM in gigabytes
attr17 false Last discovered (last_discovered) Date/Time Timestamp of the last update from primary data source provider
attr18 false Memory (MB) (memory) Integer VM memory size in megabytes
attr19 false Network adapters (nics) Integer Number of network interface adapters
attr20 false State (state) String (40) VM power state
attr21 false CI owner (supported_by) Reference (User) Responsible person for CI content (represents a CI Owner role)
attr22 false TERP Task ID Customer Project & Customer Task Unique ID used to populate dependend attributes Customer project & task in CI
attr23 false Billable (u_billable) True/False Flag for Smart Billing customer invoicing (entify whether CI is billable to customer or not)
attr24 false Customer invoicing group (u_customer_invoicing_group) String (18) Customer cost centre information or other grouping factor for the invoice
attr25 false Decomission Date (u_decomission_date) Date/Time Defines date when CI was decomissioned
attr26 false Environment (u_environment) Choice Describes if the CI is related to production or some other use
attr28 false VM Instance UUID (vm_instance_uuid) String (40) Unique identifier of the VM instance
company true Company (company) Reference (Company) SysID of the Company
related_ci false Dependent (child) Configuration Item Sys ID of dependent CI towards it is genereated CI relationship with type 'Runs on::Runs'
